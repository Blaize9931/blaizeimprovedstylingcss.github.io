CSS Selectors A CSS selector is used to identify which tag you want to make changes to. To allow for a more refined editing, you can use distinguishers Element Selector: Selects all elements of a given type (e.g., p {} for all paragraphs).
ID Selector: Selects a single element with a specific ID (e.g., #my-id {}).

Class Selector: Selects all elements with a specific class (e.g., .my-class {}).

Attribute Selector: Selects elements based on attributes (e.g., [type="text"] {}).

Pseudo-class Selector: Selects elements based on their state (e.g., a:hover {} for links on hover).

Pseudo-element Selector: Selects a part of an element (e.g., p::first-line {} for the first line of paragraphs).

Descendant Selector: Selects elements that are descendants of another element (e.g., div p {}).

Child Selector: Selects elements that are direct children of another element (e.g., ul > li {}).

Adjacent Sibling Selector: Selects an element that is immediately preceded by a specified element (e.g., h1+ p {}).

General Sibling Selector: Selects all sibling elements (e.g., h1 ~ p {}).
HTML Tags
An HTML tag consists of content, opening, and closing parameters. Both of these build what's known as an element
.		 Definition: An HTML tag is a piece of code that is used to define elements within an HTML document.
Syntax: Tags are enclosed in angle brackets. Most HTML elements consist of an opening tag and a closing tag, with content in between.

Types: Tags can be paired, or self-closing.

Attributes: Tags often have attributes that provide additional information about an element, such as class, id, style, etc.

Purpose: They help structure the content on a webpage by defining headings, paragraphs, links, images, and other elements.
Fonts I have learnt about using font-face, and linking fonts. The CSS @font-face rule allows external fonts or font files to be imported directly into stylesheets.The location of the font file must be specified in the CSS rule so that the files can be loaded from that location. This rule also allows locally hosted fonts to be added using a relative file path instead of a web URL.
Linking fonts allow user to use web fonts in the document. They can be imported in an HTML document by using the link tag. Once the web font URL is placed within the href attribute, the imported font can then be used in CSS declaration.
